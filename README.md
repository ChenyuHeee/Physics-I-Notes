## 使用方式

1. **按主题复习**：每份笔记独立覆盖 2–5 章，可直接阅读
2. **按章节查找**：笔记内部按 `N.N 章名` 组织，通过标题快速定位
3. **公式速查**：大部分笔记末尾附有公式速查表，适合考前快速回顾
4. **配图辅助**：关键图示（受力分析、几何配置、曲线图）来自原始 PPT 幻灯片

## 格式约定

```markdown
# NN. 主题名              ← 主标题
## N.N 章名                ← 章节
### N.N.N 小节名           ← 知识点分组
#### 例题 N.N.N：题名      ← 例题
```

所有数学公式使用 LaTeX 语法（`$...$` 行内、`$$...$$` 块），在 GitHub 上自动渲染。
